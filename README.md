## Getting Started

1. Clone the repository
````bash
git clone <https://github.com/KHAJAMOINUDDINKHADRI/dental-ui>
cd dental-ui
````

2. Install dependencies
````bash
npm install
# or
yarn install
````
3. Start the development server
````bash
npm run dev
# or
yarn dev
````
4. Open your browser and visit deployed site
